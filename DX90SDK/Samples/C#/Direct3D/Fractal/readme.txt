Description
===========
   The Fractal sample illustrates using Fractals to generate a height map   
Path
====
   Source:     DXSDK\Samples\C#\Direct3D\Fractal
   Executable: DXSDK\Samples\C#\Direct3D\Bin


User's Guide
============
   The following keys are implemented. The dropdown menus can be used for the
   same controls.
      <Enter>     Starts and stops the scene
      <Space>     Advances the scene by a small increment
      <F1>        Shows help or available commands.
      <F2>        Prompts user to select a new rendering device or display mode
      <Alt+Enter> Toggles between fullscreen and windowed modes
      <Esc>       Exits the app.
      <F>         Generates new Fractal
	

Programming Notes
=================
   This sample makes use of common DirectX code (consisting of helper functions,
   etc.) that is shared with other samples on the DirectX SDK. All common
   source code can be found in the following directory:
      DXSDK\Samples\C#\Common
