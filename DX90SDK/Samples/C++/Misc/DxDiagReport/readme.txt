Description
===========
   The DxDiagReport sample shows how to use the DxDiag COM interface by 
   the instantiating a known set child container by name and enumerating the 
   properties on each child container by name and stores this information
   in predefined structures.
   

Path
====
   Source:     DXSDK\Samples\C++\Misc\DxDiagReport
   Executable: DXSDK\Samples\C++\Misc\Bin


User's Guide
============
   Simply run the sample and it will enumerate and display all information 
   to a dialog box.
