Description
===========
  The Crossfade sample application shows how to fade between two audiopaths 
  
Path
====
  Source:     DXSDK\Samples\C++\DirectMusic\CrossFade
  Executable: DXSDK\Samples\C++\DirectMusic\Bin

User's Guide
============
  Click "Browse" and load a wave, MIDI, or DirectMusic Producer segment 
  file for each segment. Click "Play", and then click "Crossfade" to fade 
  between the 2 segments.
