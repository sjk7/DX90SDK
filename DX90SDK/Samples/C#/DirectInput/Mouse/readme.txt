Description
===========
  The Mouse sample obtains and displays Mouse data.

Path
====
  Source:     DXSDK\Samples\C#\DirectInput\Mouse
  Executable: DXSDK\Samples\C#\DirectInput\Bin

User's Guide
============
  Observe how the displayed data changes when you move the mouse.

Programming Notes
=================
  The application polls the Mouse for immediate data in response to a timer 
  set inside the application.
