Description
===========
  The Keyboard sample obtains and displays Keyboard data.

Path
====
  Source:     DXSDK\Samples\VB.NET\DirectInput\Keyboard
  Executable: DXSDK\Samples\VB.NET\DirectInput\Bin

User's Guide
============
  Observe how the displayed data changes when you press keys.

Programming Notes
=================
  The application polls the Keyboard for immediate data in response to a timer 
  set inside the application.
